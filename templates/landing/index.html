Platform capabilities

Everything you need.
Nothing you don't.

A full infrastructure control plane purpose-built for Odoo — from SSH provisioning to billing enforcement.

One-click Provisioning

Spin up a DigitalOcean droplet with Ubuntu 22.04, Docker, and firewall rules pre-configured — all from the dashboard.

$ dafeapp provision --region ams3 --size s-2vcpu-4gb

Bring Your Own Server

Already have a VPS? Connect any server via SSH (key or password). DafeApp verifies the connection and preps Docker automatically.

ssh-key password ed25519 rsa

Encrypted Credentials

SSH keys, passwords, and API tokens are Fernet-encrypted at field level before hitting the database. Never stored in plaintext.

AES-128 · Fernet · FIELD_ENCRYPTION_KEY

Role-based Team Access

Four-tier RBAC: Super Admin, Admin, Manager, User. Each organization has its own isolated workspace with invitation-based onboarding.

Super Admin Admin Manager User

Full Audit Trail

Every action — logins, server additions, deployments, role changes — is logged with user, IP, and timestamp. Built for compliance from day one.

09:12 admin@co server_add prod-do-01
09:15 admin@co server_verify OK
09:18 admin@co server_prepare done

Subscription & Billing

Plan-based feature gates with instance limits, backup quotas, staging access, and version upgrade controls — all enforced at the service layer.
